## 一、整體版面結構概覽

後台採用 **左側 Sidebar + 上方 Header + 主內容區** 的經典後台 layout，適用於桌面與平板；手機版改為 **頂部 Header + 可收合 Sidebar（Drawer）**。

```
┌─────────────────────────────────────────────────────────────┐
│  Header（固定頂端）                                            │
│  麵包屑 / 頁面標題 │ 快捷操作 │ 前台預覽 │ 使用者（未來）       │
├──────────────┬──────────────────────────────────────────────┤
│              │                                                │
│   Sidebar    │                                                │
│   （固定左側）│            Main Content Area                    │
│              │            （可捲動）                           │
│   - Logo     │                                                │
│   - 導航連結  │                                                │
│   - 區塊分組  │                                                │
│              │                                                │
└──────────────┴─────────────────────────────────────────────┘
```

## 二、Sidebar 結構

### 2.1 區塊劃分

Sidebar 分為三區：

| 區塊 | 位置 | 內容 |
|------|------|------|
| **品牌區** | 最上方 | Logo、網站名稱或「後台」字樣，可點擊回 Dashboard |
| **主導航區** | 中間 | 各管理模組連結 |
| **底區** | 最下方 | 設定類、前台連結、未來登出 |

### 2.2 主導航分組結構

```
【主導航】
├── 📊 總覽              → /admin
│
├── 📚 課程                （可展開/收合）
│   ├── 課程列表           → /admin/courses
│   └── 課程分類           → /admin/categories
│
├── 🏠 首頁                → /admin/home
├── 🖼️ Banner             → /admin/banners
│
├── 📄 內容                （可展開/收合）
│   ├── 成果案例           → /admin/stories
│   ├── FAQ               → /admin/faqs
│   └── 最新消息           → /admin/news
│
├── ℹ️ 關於我們            → /admin/about
│
└── ⚙️ 設定                （可展開/收合）
    ├── 網站設定           → /admin/settings
    └── SEO 設定           → /admin/seo
```

### 2.3 分組設計原則

| 原則 | 說明 |
|------|------|
| **課程** | 課程與分類為核心，獨立一組 |
| **內容** | 案例、FAQ、消息皆為內容型模組，歸為一組 |
| **設定** | 網站與 SEO 皆為設定類，歸為一組 |
| **展開狀態** | 預設展開當前頁面所屬群組；其他群組可收合以節省空間 |
| **圖示** | 每項可配簡易圖示（Emoji 或 icon），提升辨識度 |

### 2.4 Sidebar 視覺規格（建議）

| 項目 | 桌面版 | 說明 |
|------|--------|------|
| 寬度 | 240px ~ 260px | 可收合為圖示模式時約 64px |
| 背景 | 淺灰或白，與內容區有邊界線 | 區隔主內容 |
| 項目高度 | 約 40px ~ 44px | 易點擊 |
|  hover | 淺灰背景 | 明確 feedback |
| 當前頁 | 左側色條 + 背景色 | 清楚標示所在頁面 |
| 字級 | 14px ~ 15px | 易讀、不佔空間 |

### 2.5 Sidebar 收合模式（可選）

- 點擊品牌區旁按鈕或底部「收合」可切換
- 收合後僅顯示圖示，hover 可展開浮動子選單
- 初期可先不做，維持固定展開

## 三、Header 結構

### 3.1 區塊劃分

Header 為單一橫列，由左到右分為三區：

```
┌─────────────────────────────────────────────────────────────────┐
│ [☰]  首頁 > 課程管理 > 課程列表       [新增課程] [前台預覽] [···]  │
└─────────────────────────────────────────────────────────────────┘
```

| 區塊 | 位置 | 內容 |
|------|------|------|
| **左區** | 左側 | 手機版選單按鈕（☰）、麵包屑導航 |
| **中區** | 中央 | 當前頁面標題（可作為麵包屑最後一項） |
| **右區** | 右側 | 快捷操作按鈕、前台預覽、未來使用者選單 |

### 3.2 麵包屑

- 格式：`首頁 > [父層] > 當前頁`
- 範例：
  - Dashboard：`首頁`
  - 課程列表：`首頁 > 課程管理 > 課程列表`
  - 編輯課程：`首頁 > 課程管理 > 課程列表 > 編輯：課程名稱`
- 每一層可點擊回對應頁面

### 3.3 頁面標題

- 顯示當前模組或頁面名稱
- 與麵包屑最後一項可合併或分開顯示，依版面選擇

### 3.4 快捷操作（右區）

| 按鈕 | 顯示時機 | 行為 |
|------|----------|------|
| **新增** | 列表頁 | 導向 `/admin/[module]/new` 或開啟新增 Modal |
| **儲存** | 編輯/新增頁 | 送出表單 |
| **取消** | 編輯/新增頁 | 返回列表或關閉 Modal |
| **前台預覽** | 全部 | 開新分頁到對應前台頁面 |
| **更多**（···） | 可選 | 下拉選單：匯出、設定等 |

### 3.5 Header 視覺規格

| 項目 | 建議 |
|------|------|
| 高度 | 56px ~ 64px |
| 背景 | 白或極淺灰，底部邊框 |
| 固定 | 捲動時固定於頂端 |
| 陰影 | 輕微 shadow 區隔內容區 |

### 4.2 統計卡片區（StatCards）

| 項目 | 說明 |
|------|------|
| **數量** | 6 張卡片：課程總數、招生中、精選、消息、FAQ、案例 |
| **版面** | 響應式 grid：桌面 3x2 或 6 列一排，平板 2x3，手機 2x3 或 1 列 |
| **內容** | 標題（小字）、數字（大字）、可選圖示 |
| **點擊** | 可連結至對應列表頁（如「課程總數」→ `/admin/courses`） |
| **樣式** | 白底、邊框或輕陰影，與主內容區一致 |

### 4.3 快速操作區（QuickActions）

| 項目 | 說明 |
|------|------|
| **按鈕** | 新增課程、新增消息、新增案例、前往設定 |
| **樣式** | 主按鈕樣式，可橫排或 2x2 grid |
| **行為** | 導向對應 `/new` 或列表頁 |

### 4.4 最近更新區（RecentUpdates）

| 項目 | 說明 |
|------|------|
| **資料來源** | 課程、消息、案例，依 updatedAt 排序 |
| **顯示** | 類型、標題、更新時間、操作（編輯、預覽） |
| **筆數** | 建議 5 ~ 10 筆 |
| **「查看全部」** | 可連結到整合的近期更新頁，或各自列表 |
| **空狀態** | 無資料時顯示「尚無更新」 |

## 五、列表頁設計

### 5.1 通用結構

所有列表頁（課程、分類、Banner、案例、FAQ、消息）共用同一結構：

```
┌─────────────────────────────────────────────────────────────────────┐
│ [頁面標題]                                              [新增] 按鈕   │
├─────────────────────────────────────────────────────────────────────┤
│  搜尋框    [分類篩選▼] [狀態篩選▼] [排序▼]          [篩選] [清除]    │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │ ☐ │ 欄位1   │ 欄位2    │ 欄位3   │ 狀態  │ 操作            │   │
│  ├───┼─────────┼──────────┼─────────┼───────┼─────────────────┤   │
│  │ ☐ │ 資料... │ 資料...  │ ...     │ Badge │ [編輯][刪除][···]│   │
│  │ ☐ │ 資料... │ 資料...  │ ...     │ Badge │ [編輯][刪除][···]│   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                      │
│  共 12 筆                      [上一頁] 1 2 3 [下一頁]               │
└─────────────────────────────────────────────────────────────────────┘
```

### 5.2 區塊說明

| 區塊 | 內容 | 備註 |
|------|------|------|
| **工具列** | 搜尋、篩選、排序、新增 | 響應式時可收合進「篩選」按鈕 |
| **搜尋** | 單一輸入框，即時或按 Enter | 搜尋標題、摘要等主要欄位 |
| **篩選** | 依模組：分類、狀態、精選等 | 下拉或 checkbox 群組 |
| **排序** | 依日期、標題、sortOrder | 下拉選單 |
| **表格** | 欄位依模組而定，見下表 | 支援橫向捲動 |
| **操作欄** | 編輯、刪除、預覽、更多 | 固定於右側 |
| **批次操作** | 可選全選、批次刪除、批次改狀態 | 勾選後顯示 |
| **分頁** | 每頁筆數、上一頁、頁碼、下一頁 | 資料多時顯示 |

### 5.3 各模組列表欄位建議

| 模組 | 主要欄位 | 其他 |
|------|----------|------|
| **課程** | 標題、主分類、子分類、開課日期、狀態、精選 | 操作：編輯、刪除、預覽、招生切換 |
| **分類** | 名稱、父分類、順序、顯示 | 操作：編輯、刪除 |
| **Banner** | 標題、排序、啟用 | 操作：編輯、刪除、上下移 |
| **案例** | 標題、分類、精選、發布日期 | 操作：編輯、刪除、預覽 |
| **FAQ** | 問題、分類、首頁顯示、排序 | 操作：編輯、刪除 |
| **消息** | 標題、類型、發布日期、發布狀態 | 操作：編輯、刪除、預覽 |

### 5.4 表格互動

| 項目 | 說明 |
|------|------|
| **hover** | 行 hover 時淺色背景 |
| **空狀態** | 無資料時顯示圖示 + 「尚無資料，點擊新增」 |
| **載入** | Skeleton 或 loading 提示 |
| **刪除** | 點刪除 → 確認對話框 → 確認後刪除 |

## 七、手機版處理方式

### 7.1 版面適應策略

| 斷點 | 策略 |
|------|------|
| **≥ 1024px** | 完整 Sidebar + Header + 內容 |
| **768px ~ 1023px** | Sidebar 可收合為圖示或隱藏，以按鈕開啟 |
| **< 768px** | 頂部 Header + Drawer 式 Sidebar，內容全寬 |

### 7.2 手機版 Sidebar

| 項目 | 說明 |
|------|------|
| **預設** | 隱藏，不佔空間 |
| **觸發** | 點 Header 左側 ☰ 開啟 |
| **呈現** | 左側滑出 Drawer，寬度約 80% 或 280px |
| **背景** | 打開時背後半透明遮罩，點遮罩關閉 |
| **內容** | 與桌面相同導航結構，可垂直捲動 |

### 7.3 手機版 Header

| 項目 | 說明 |
|------|------|
| **左** | ☰ 選單按鈕（固定） |
| **中** | 頁面標題（可縮短或省略） |
| **右** | 主要操作（如新增、儲存），其餘收進 ⋯ 選單 |

### 7.4 手機版 Dashboard

| 區塊 | 處理 |
|------|------|
| **統計卡片** | 2 列 3 行 grid，或單列可橫向捲動 |
| **快速操作** | 2x2 grid 或單列捲動 |
| **最近更新** | 改為卡片列表，每筆一行，操作改為圖示或長按 |

### 7.5 手機版列表頁

| 項目 | 說明 |
|------|------|
| **工具列** | 搜尋置頂，篩選/排序收進「篩選」按鈕，點擊開啟底部 Sheet 或全屏 |
| **表格** | 改為卡片列表，每筆一卡，顯示關鍵欄位 |
| **操作** | 每卡底部：編輯、刪除，或右滑顯示操作 |
| **分頁** | 改為「載入更多」或簡化分頁 |

### 7.6 手機版編輯頁

| 項目 | 說明 |
|------|------|
| **分區** | 維持 FormSection，改為 Accordion 可摺疊，減輕捲動負擔 |
| **欄位** | 單欄、全寬，避免多欄 |
| **陣列輸入** | 如 outline、FAQ，改為可展開的區塊，每筆一區 |
| **儲存/取消** | 固定於底部 Sticky Bar，避免被鍵盤遮擋 |

### 7.7 觸控與操作

| 項目 | 說明 |
|------|------|
| **點擊區域** | 至少 44x44px |

# Class Demo - 職訓課程平台

職訓課程彙整網站，協助待業者、轉職者、在職進修者找到適合的政府補助與自費課程。

## 功能特色

- 📚 **課程瀏覽** - 政府補助課程與自費課程分類展示
- 🔍 **課程篩選** - 依類別、類型篩選課程
- 📋 **課程詳情** - 課程介紹、FAQ、報名資訊
- 📄 **其他頁面** - 關於我們、常見問題、報名流程

## 技術棧

- [Next.js 14](https://nextjs.org/) - React 框架
- [React 18](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## 專案結構

```
src/
├── app/           # Next.js App Router 頁面
├── components/    # React 元件
├── lib/           # 工具函式與資料層
├── data/          # JSON 靜態資料
└── types/         # TypeScript 型別定義
```

## 快速開始

### 環境需求

- Node.js 18+
- npm 或 pnpm

### 安裝依賴

```bash
npm install
```

### 開發模式

```bash
npm run dev
```

開啟 [http://localhost:3000](http://localhost:3000) 預覽網站。

### 建置

```bash
npm run build
```

### 正式環境執行

```bash
npm run start
```

### 程式碼檢查

```bash
npm run lint
```

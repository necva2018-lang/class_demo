# 職訓課程網站 — 產品規格文件（Product Spec）

> 版本：v1.0 | 建立日期：2025-03-15

---

## 1. 產品定位

| 項目 | 說明 |
|------|------|
| **產品名稱** | 職訓課程網站 |
| **核心價值主張** | 以招生轉換為核心，讓使用者快速找到適合的課程並完成報名 |
| **定位描述** | 一站式職訓課程資訊與報名平台，串接政府補助與自費課程，協助各類求職／進修者找到合適培訓並完成報名 |
| **與競品差異** | 聚焦「找課 → 了解 → 報名」的轉換流程，減少搜尋與報名摩擦 |

---

## 2. 使用者目標

| 受眾類型 | 主要需求 | 期望結果 |
|----------|----------|----------|
| **待業者** | 快速取得就業技能、找到有補助的職前訓練 | 順利結訓、媒合工作 |
| **轉職者** | 跨領域技能培訓、政府補助減輕負擔 | 成功轉職、提升薪資 |
| **二度就業者** | 重返職場所需技能、彈性時間的課程 | 重返職場、適應新環境 |
| **在職進修者** | 進修、證照、加值技能 | 職涯加分、升遷機會 |
| **青年培訓對象** | 職涯探索、政府專班、青年專屬方案 | 累積實力、職涯起步 |

## 3. 網站核心功能

### 3.1 課程搜尋與篩選
- 依課程分類（政府補助／自費）篩選
- 依訓練類型（職前／在職／專班／證照／推廣）篩選
- 依開課時間、地點、時數、費用篩選
- 關鍵字搜尋

### 3.2 課程資訊展示
- 課程分類階層明確
- 課程列表（卡片或清單）
- 課程詳情（大綱、師資、時數、費用、補助說明、開課日期）

### 3.3 招生轉換
- 線上報名表單
- 報名進度查詢（可規劃第二階段）
- CTA 置於關鍵頁面，引導至報名流程

### 3.4 信任建立
- 成果案例展示
- 最新消息與公告
- FAQ 常見問題
- 關於我們（品牌與團隊介紹）

### 3.5 導航與體驗
- 清楚網站結構
- 響應式設計（RWD）
- 快速載入與流暢操作

## 5. 每頁功能摘要

### 5.1 首頁
| 區塊 | 功能說明 |
|------|----------|
| **主視覺／主橫幅** | 核心價值主張、主 CTA（例如：立即找課程） |
| **熱門／推薦課程** | 精選課程卡片，可連結至課程詳細頁 |
| **課程分類入口** | 政府補助／自費課程兩大類，點擊進入分類頁 |
| **成果數據** | 結訓人數、就業率、合作企業等數字，提升信任 |
| **最新消息精選** | 2–3 則最新公告或消息 |
| **報名入口** | 顯眼 CTA 按鈕，引導至報名頁或課程總覽 |
| **頁尾** | 聯絡資訊、社群連結、快速連結 |

### 5.2 課程總覽頁
| 區塊 | 功能說明 |
|------|----------|
| **麵包屑** | 首頁 > 課程總覽 |
| **篩選器** | 課程分類、訓練類型、開課時間、地點、費用區間等 |
| **排序** | 依開課日期、人氣、費用等排序 |
| **課程列表** | 課程卡片（圖片、標題、簡述、費用、開課日期、報名按鈕） |
| **分頁或載入更多** | 支援大量課程瀏覽 |
| **空狀態** | 無符合條件時提示調整篩選 |

### 5.3 課程分類頁
| 區塊 | 功能說明 |
|------|----------|
| **分類導航** | 政府補助（職前／在職／專班）、自費（證照／推廣） |
| **分類說明** | 簡述該類課程特色與適用對象 |
| **課程列表** | 與總覽頁同風格，僅預設套用分類篩選 |
| **篩選與排序** | 與總覽頁邏輯一致 |

### 5.4 課程詳細頁
| 區塊 | 功能說明 |
|------|----------|
| **課程標題與標籤** | 課程名稱、分類、補助狀態 |
| **主要資訊** | 開課日期、時數、地點、費用、名額 |
| **課程簡介** | 課程目標、適合對象 |
| **課程大綱** | 單元與內容說明 |
| **師資介紹** | 講師經歷與專長 |
| **補助說明** | 政府補助資格、金額、申請方式 |
| **報名 CTA** | 固定或浮動「立即報名」按鈕 |
| **相關課程推薦** | 同分類或同類型課程 |
| **分享功能** | 分享至社群或複製連結（可選） |

### 5.5 成果案例頁
| 區塊 | 功能說明 |
|------|----------|
| **頁面標題與簡介** | 說明成果案例的價值 |
| **案例篩選** | 依班別、結訓年度、產業類型等（可選） |
| **案例卡片／列表** | 學員故事、就業成果、企業合作、數據統計 |
| **圖文或影音** | 學員見證、成果照片、數據圖表 |
| **CTA** | 引導至課程總覽或報名 |

### 5.6 最新消息頁
| 區塊 | 功能說明 |
|------|----------|
| **消息列表** | 標題、摘要、日期、分類標籤 |
| **篩選／分類** | 依類型（公告、課程訊息、活動等）篩選 |
| **分頁** | 支援多頁瀏覽 |
| **點擊進入** | 可連結至消息詳細頁（可規劃） |

### 5.7 關於我們頁
| 區塊 | 功能說明 |
|------|----------|
| **機構介紹** | 成立背景、使命、願景 |
| **服務特色** | 課程品質、師資、合作單位 |
| **聯絡資訊** | 電話、Email、地址、營業時間 |
| **交通資訊** | 地圖或大眾運輸說明（可選） |
| **組織架構** | 依需求決定是否展示 |

### 5.8 FAQ 頁
| 區塊 | 功能說明 |
|------|----------|
| **分類導覽** | 報名流程、補助申請、課程相關、其他 |
| **問答列表** | 可收合／展開的 Accordion |
| **搜尋 FAQ** | 關鍵字搜尋問題（可選） |
| **聯絡引導** | 找不到解答時，引導至聯絡表單或客服 |

### 5.9 報名頁
| 區塊 | 功能說明 |
|------|----------|
| **課程確認** | 顯示所選課程名稱與基本資訊 |
| **報名表單** | 姓名、聯絡方式、身分證字號、學歷、經歷、報名動機等 |
| **欄位驗證** | 必填、格式檢查、送出前確認 |
| **送出與確認** | 送出後顯示成功訊息、報名序號或查詢方式 |
| **注意事項** | 補助資格、應備文件、聯絡窗口 |

## 6. 建議的元件列表

### 6.1 全域元件
| 元件名稱 | 用途 |
|----------|------|
| **Header** | 全站導覽、Logo、主選單、行動版選單 |
| **Footer** | 聯絡資訊、快速連結、版權宣告 |
| **麵包屑 (Breadcrumb)** | 層級導航 |
| **CTA 按鈕** | 統一樣式，如「立即報名」「查看課程」 |
| **聯絡資訊區塊** | 電話、Email、Line 等 |

### 6.2 首頁元件
| 元件名稱 | 用途 |
|----------|------|
| **Hero Banner** | 主視覺、主標、副標、主 CTA |
| **課程分類入口卡** | 政府補助／自費課程入口 |
| **課程卡片 (CourseCard)** | 課程預覽，含標題、摘要、費用、報名按鈕 |
| **數據統計區塊** | 數字＋文案，建立信任 |
| **消息精選列表** | 最新 2–3 則消息 |

### 6.3 課程相關元件
| 元件名稱 | 用途 |
|----------|------|
| **課程篩選器 (Filter)** | 分類、類型、日期、地點、費用 |
| **課程列表 (CourseList)** | 多個 CourseCard 組合 |
| **課程詳情區塊** | 標題、資訊、大綱、師資、補助說明 |
| **課程標籤 (Tag)** | 分類、補助、熱門等 |
| **報名按鈕 (Floating/Sticky)** | 固定於畫面，方便報名 |

### 6.4 內容頁元件
| 元件名稱 | 用途 |
|----------|------|
| **案例卡片 (CaseCard)** | 成果案例摘要 |
| **消息列表項目** | 標題、摘要、日期、連結 |
| **FAQ Accordion** | 問答可收合區塊 |
| **圖文區塊** | 關於我們、機構介紹 |

### 6.5 表單元件
| 元件名稱 | 用途 |
|----------|------|
| **報名表單** | 表單容器、驗證、送出 |
| **表單輸入 (Input)** | 文字、下拉、多選、日期等 |
| **表單驗證訊息** | 錯誤提示、成功訊息 |
| **送出成功區塊** | 報名成功頁或區塊 |

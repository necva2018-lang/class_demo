# 職訓課程網站 — UI/UX Wireframe 結構規劃

> 以招生轉換率高為優先 | 文字版 Wireframe Spec

---

## 共通原則

- **轉換漏斗**：首頁 → 課程總覽 → 課程詳情 → 報名頁，每層都要有明確 CTA
- **信任優先**：成果、數據、FAQ 在關鍵決策前出現
- **減少摩擦**：表單精簡、步驟少、手機操作順暢
- **行動優先**：設計以手機版為基準，再擴到桌面版

---

## 1. 首頁

### 1.1 頁面目標
- 讓訪客快速理解「這是職訓課程平台」並找到適合的課程入口
- 建立初步信任，降低跳出率
- 引導至課程總覽或分類頁，或直接報名

### 1.2 主要區塊順序

| 順序 | 區塊名稱 | 內容 |
|------|----------|------|
| 1 | Header | Logo、主選單（課程、成果、最新消息、關於我們、FAQ）、報名入口按鈕、手機版漢堡選單 |
| 2 | Hero | 主標（例：找到適合你的職訓課程）、副標（簡述價值）、搜尋列或分類按鈕、主 CTA（立即找課程 或 政府補助／自費入口） |
| 3 | 熱門／推薦課程 | 3–4 張課程卡片，每張含：課程名稱、分類標籤、開課日期、費用、簡述、報名按鈕 |
| 4 | 課程分類入口 | 兩大塊：政府補助課程、自費課程，各含子分類（職前／在職／專班、證照／推廣），點擊進入對應分類頁 |
| 5 | 成果數據 | 數字區塊：例「累計結訓 X 人」「就業率 X%」「合作企業 X 家」，建立信任 |
| 6 | 最新消息精選 | 2–3 則最新消息，標題＋日期＋摘要，點擊可進詳細 |
| 7 | CTA 區塊 | 強烈引導：例「還不確定選哪堂？先看課程總覽」＋按鈕 |
| 8 | Footer | 聯絡資訊、快速連結、關於我們、社群、版權 |

### 1.3 各區塊內容重點

| 區塊 | 內容 |
|------|------|
| Hero | 主標簡短有力；副標說明補助、就業、進修；搜尋列可選填關鍵字後跳課程總覽；或直接給分類按鈕 |
| 熱門課程 | 選開課近、名額多、補助高的課程；卡片資訊精簡，突出「報名」 |
| 分類入口 | 視覺上兩塊分明，每塊列出子分類名稱，讓使用者一眼辨識 |
| 成果數據 | 具體數字，可搭配圖示或簡短文案 |
| 最新消息 | 標題＋日期即可，減少干擾 |

### 1.4 各區塊 CTA 建議

| 區塊 | CTA |
|------|-----|
| Header | 「我要報名」按鈕 |
| Hero | 「立即找課程」「政府補助課程」「自費課程」 |
| 熱門課程 | 每張卡片「報名」或「查看詳情」 |
| 分類入口 | 各子分類為可點擊按鈕／卡片 |
| 成果數據 | 可選加「看更多成果」連結 |
| 最新消息 | 每則「閱讀更多」 |
| CTA 區塊 | 「查看全部課程」或「立即報名」 |

### 1.5 手機版注意

- Hero 主標縮短，副標可折行
- 熱門課程改為單列滑動或 1 張一屏
- 分類入口改為垂直堆疊或 Tab 切換
- 成果數據可橫向滑動或 2×2 網格
- Footer 精簡為摺疊式或只留主要連結
- Header 報名按鈕要顯眼、易點

## 2. 課程總覽頁

### 2.1 頁面目標
- 讓使用者快速篩選、找到有興趣的課程
- 降低選擇困難，提高進入課程詳情頁的比例

### 2.2 主要區塊順序

| 順序 | 區塊名稱 | 內容 |
|------|----------|------|
| 1 | Header | 同首頁 |
| 2 | 麵包屑 | 首頁 > 課程總覽 |
| 3 | 頁面標題 | 例：「所有課程」或依分類顯示「政府補助／職前訓練」 |
| 4 | 篩選區 | 篩選器：課程分類、訓練類型、開課月份、地點、費用區間、關鍵字搜尋；排序：開課日期、費用、人氣 |
| 5 | 課程列表 | 課程卡片列表，每張：圖片、標題、分類標籤、開課日期、時數、地點、費用、報名按鈕 |
| 6 | 分頁／載入更多 | 分頁或「載入更多」按鈕 |
| 7 | 無結果狀態 | 無符合條件時提示調整篩選＋「清除篩選」按鈕 |
| 8 | CTA 浮動區（可選） | 手機版底部固定「需要協助？聯絡我們」 |
| 9 | Footer | 同首頁 |

### 2.3 各區塊內容重點

| 區塊 | 內容 |
|------|------|
| 篩選區 | 手機版可摺疊為「篩選」按鈕，點開為全屏或抽屜；桌面版可側欄或頂部橫排 |
| 課程列表 | 預設顯示符合篩選的課程，每張卡片資訊一致、可掃描 |
| 無結果 | 友善提示＋建議動作（改篩選或聯絡我們） |

### 2.4 各區塊 CTA 建議

| 區塊 | CTA |
|------|-----|
| 課程卡片 | 「查看詳情」「立即報名」 |
| 無結果 | 「清除篩選」「聯絡我們」 |
| 浮動區 | 「聯絡我們」「Line 諮詢」 |

### 2.5 手機版注意

- 篩選改為抽屜或全屏，篩選結果數量要顯示
- 課程卡片改為單列、垂直滑動，資訊精簡
- 報名按鈕夠大、易點，避免誤觸
- 分頁可改「載入更多」減少捲動
- 篩選與排序可合併為一個入口

## 3. 課程詳細頁

### 3.1 頁面目標
- 提供完整課程資訊，消除疑慮
- 促成「報名」決定，提高轉換

### 3.2 主要區塊順序

| 順序 | 區塊名稱 | 內容 |
|------|----------|------|
| 1 | Header | 同首頁 |
| 2 | 麵包屑 | 首頁 > 課程 > [分類] > [課程名稱] |
| 3 | 課程標題區 | 課程名稱、分類標籤（政府補助／自費、職前／在職等）、補助狀態（例：全額補助） |
| 4 | 關鍵資訊摘要 | 開課日期、時數、地點、費用、剩餘名額、報名截止日，以 icon＋文字呈現 |
| 5 | 主要 CTA | 明顯「立即報名」按鈕 |
| 6 | 課程簡介 | 課程目標、適合對象、預期成果 |
| 7 | 課程大綱 | 單元列表：單元名稱、內容說明、時數 |
| 8 | 師資介紹 | 講師姓名、經歷、專長、照片（可選） |
| 9 | 補助說明（有補助時） | 補助資格、金額、申請方式、應備文件 |
| 10 | 相關課程推薦 | 2–4 張同分類或同類型課程卡片 |
| 11 | 固定／浮動報名 CTA | 手機版底部固定「立即報名」 |
| 12 | Footer | 同首頁 |

### 3.3 各區塊內容重點

| 區塊 | 內容 |
|------|------|
| 關鍵資訊 | 一眼可見的決策資訊，政府補助要顯著標示 |
| 課程大綱 | 可摺疊或分區呈現，避免過長 |
| 補助說明 | 資格清楚、步驟簡短，減少疑慮 |
| 相關課程 | 防止無合適課程時流失，可引導至其他課程 |

### 3.4 各區塊 CTA 建議

| 區塊 | CTA |
|------|-----|
| 標題區下方 | 「立即報名」主按鈕 |
| 相關課程 | 每張卡片「查看詳情」「報名」 |
| 浮動區 | 「立即報名」 |
| 補助說明區 | 可加「我有資格嗎？聯絡諮詢」 |

### 3.5 手機版注意

- 關鍵資訊改為 2 列或垂直堆疊
- 報名按鈕固定底部，始終可見
- 課程大綱、師資、補助可摺疊，減少捲動
- 相關課程可橫向滑動
- 麵包屑可簡化或縮短

## 4. 關於我們頁

### 4.1 頁面目標
- 建立品牌信任，讓訪客了解機構背景
- 提供聯絡方式，方便猶豫者諮詢

### 4.2 主要區塊順序

| 順序 | 區塊名稱 | 內容 |
|------|----------|------|
| 1 | Header | 同首頁 |
| 2 | 麵包屑 | 首頁 > 關於我們 |
| 3 | 頁面標題 | 例：「關於我們」 |
| 4 | 機構簡介 | 成立背景、使命、願景、核心價值，圖文並茂 |
| 5 | 服務特色 | 3–4 項：例課程品質、師資、就業輔導、合作企業，每項簡短圖示＋文案 |
| 6 | 聯絡資訊 | 電話、Email、地址、營業時間、Line（如有） |
| 7 | 交通／地圖（可選） | 簡圖或嵌入地圖、大眾運輸資訊 |
| 8 | CTA 區 | 「查看課程」「立即報名」或「聯絡我們」 |
| 9 | Footer | 同首頁 |

### 4.3 各區塊內容重點

| 區塊 | 內容 |
|------|------|
| 機構簡介 | 約 2–3 段，重點放在「我們能幫助你什麼」 |
| 服務特色 | 與招生相關：就業率、補助、師資、實作 |
| 聯絡資訊 | 清楚、易複製，可一鍵撥號、開 Email |

### 4.4 各區塊 CTA 建議

| 區塊 | CTA |
|------|-----|
| 服務特色後 | 「查看課程」 |
| 聯絡資訊 | 電話、Email、Line 可點擊 |
| CTA 區 | 「查看課程」「立即報名」「聯絡我們」 |

### 4.5 手機版注意

- 服務特色改為垂直堆疊或滑動
- 電話、Email 使用 `tel:`、`mailto:` 方便點擊
- 地圖可縮小或改為「開啟地圖」連結
- CTA 區按鈕夠大、易點

## 5. 成果案例頁

### 5.1 頁面目標
- 以真實案例建立信任
- 降低報名顧慮，引導至課程或報名

### 5.2 主要區塊順序

| 順序 | 區塊名稱 | 內容 |
|------|----------|------|
| 1 | Header | 同首頁 |
| 2 | 麵包屑 | 首頁 > 成果案例 |
| 3 | 頁面標題＋簡介 | 例：「學員成果」＋一兩句說明 |
| 4 | 整體數據 | 例：累計結訓人數、就業率、合作企業數 |
| 5 | 案例篩選（可選） | 依班別、年度、產業類型篩選 |
| 6 | 案例列表 | 案例卡片：標題、學員背景、成果、照片／圖表、相關課程 |
| 7 | CTA 區 | 「找到適合你的課程」「立即報名」 |
| 8 | Footer | 同首頁 |

### 5.3 各區塊內容重點

| 區塊 | 內容 |
|------|------|
| 整體數據 | 數字清晰，搭配簡短說明 |
| 案例卡片 | 學員背景＋改變＋結果；可連結到相關課程 |

### 5.4 各區塊 CTA 建議

| 區塊 | CTA |
|------|-----|
| 整體數據後 | 「查看課程」 |
| 案例卡片 | 「相關課程」「查看課程」 |
| CTA 區 | 「查看課程」「立即報名」 |

### 5.5 手機版注意

- 整體數據可 2×2 或橫向滑動
- 案例卡片單列，可摺疊詳情
- 案例多時可載入更多或分頁
- CTA 區按鈕明顯

## 6. FAQ 頁

### 6.1 頁面目標
- 回答常見問題，減少疑慮
- 找不到答案時導向聯絡或報名

### 6.2 主要區塊順序

| 順序 | 區塊名稱 | 內容 |
|------|----------|------|
| 1 | Header | 同首頁 |
| 2 | 麵包屑 | 首頁 > 常見問題 |
| 3 | 頁面標題＋簡介 | 例：「常見問題」＋簡短說明 |
| 4 | 分類導覽（可選） | 報名流程、補助申請、課程相關、其他，可錨點或 Tab |
| 5 | FAQ 列表 | Accordion：問題＋答案，可搜尋（可選） |
| 6 | 找不到答案區 | 例：「還有其他問題？」＋聯絡方式、Line、表單 |
| 7 | CTA 區 | 「查看課程」「立即報名」 |
| 8 | Footer | 同首頁 |

### 6.3 各區塊內容重點

| 區塊 | 內容 |
|------|------|
| FAQ 列表 | 報名流程、補助資格、上課方式、退款等常見疑問 |
| 找不到答案 | 聯絡資訊清楚，可加簡易表單或 Line |

### 6.4 各區塊 CTA 建議

| 區塊 | CTA |
|------|-----|
| 分類導覽 | 點擊跳至對應區塊 |
| FAQ 答案內 | 可加「查看課程」「立即報名」連結 |
| 找不到答案 | 「聯絡我們」「Line 諮詢」「填寫表單」 |
| CTA 區 | 「查看課程」「立即報名」 |

### 6.5 手機版注意

- FAQ Accordion 單次只開一個，避免過長
- 分類導覽可改為下拉或 Tab
- 聯絡資訊可一鍵撥號、開 Line
- CTA 區按鈕夠大

## 7. 報名頁

### 7.1 頁面目標
- 完成報名表單，產生有效報名
- 降低表單放棄率，減少錯誤填寫

### 7.2 主要區塊順序

| 順序 | 區塊名稱 | 內容 |
|------|----------|------|
| 1 | Header | Logo＋返回課程連結，手機版可精簡 |
| 2 | 麵包屑 | 首頁 > 課程 > [課程名稱] > 報名 |
| 3 | 課程確認區 | 課程名稱、開課日期、地點、費用、補助說明（若有），讓使用者確認 |
| 4 | 報名表單 | 分區：基本資料（姓名、電話、Email、身分證）、背景（學歷、經歷）、報名動機、備註 |
| 5 | 注意事項 | 應備文件、補助資格、聯絡窗口、隱私說明 |
| 6 | 送出區 | 送出按鈕、「我同意服務條款」勾選（如有） |
| 7 | 送出成功區 | 成功訊息、報名序號、後續步驟說明、可選「返回課程」 |
| 8 | Footer | 可精簡，或只留聯絡方式 |

### 7.3 各區塊內容重點

| 區塊 | 內容 |
|------|------|
| 課程確認 | 一眼看清報名課程，可編輯或返回更改 |
| 表單 | 必填標示清楚、欄位分組、驗證即時回饋 |
| 注意事項 | 簡短、易讀，避免過長法律條文 |
| 送出成功 | 明確成功＋序號＋下一步，減少焦慮 |

### 7.4 各區塊 CTA 建議

| 區塊 | CTA |
|------|-----|
| 課程確認 | 「更換課程」連結 |
| 表單底部 | 「送出報名」主按鈕 |
| 送出成功 | 「返回課程」「查看其他課程」 |

### 7.5 手機版注意

- 表單單欄、欄位夠大、鍵盤友善（適當 input type）
- 送出按鈕固定底部或明顯位置
- 課程確認區精簡，可摺疊
- 注意事項可摺疊，減少干擾
- 送出成功區序號易複製（可點擊複製）
